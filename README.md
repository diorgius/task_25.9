# Продвинутый Backend. Модуль 25. Работа с БД.

Практическая работа

## Задание 25.9

### В рамках данной практической работы создана галерея изображений в которой реализовано:
- Регистрация и авторизация пользователей.
- Загрузка файлов авторизованными пользователями.
- Вывод галереи загруженных изображений.
- Удаление загруженных файлов авторизованными пользователями с удалением комментариев к удаляемому изображению.
- Возможность оставлять комментарии к изображению авторизованными пользователями.
- Авторизованный пользователь может удалить свой комментарий.
- Неавторизованный пользователь имеет возможность только просмотра галереи и комментариев к изображениям.
